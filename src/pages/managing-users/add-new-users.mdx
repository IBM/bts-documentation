<PageDescription>

Only a registered **Account Admin** can add new human users to your **organization**.
In the IBM Blockchain Transparent Supply™ **Users** module, the administrator enters each new user's
name and **email address** and assigns user roles.

</PageDescription>

**Note:** Adding a new system user requires registering a system ID; see [Authenticate
system users](/managing-users/authenticate-system-users) for details.

## Get started

**Attention:** Each new human user must have a [**registered IBMid**](https://www.ibm.com/account/us-en/signup/register.html)
in order to log in to IBM Blockchain Transparent Supply™. The email address used to register their
IBMid must match their IBM Blockchain Transparent Supply™ user ID (email address). If necessary, you
can change a registered IBM Blockchain Transparent Supply™ user ID to match the user's IBMid.

Complete the following steps to add new users to your organization.

 1. Log in to <a href={process.env.GATSBY_UI_URL + '/'} target="_blank">IBM Blockchain Transparent Supply™</a> as an **Account Admin**.
 1. Select the <a href={process.env.GATSBY_UI_URL + '/manageUsers'} target="_blank">**Users**</a> module.
 1. Select the **Account users** tab.
 1. Click **Register user**
 1. Enter the new user's name, IBMid email, and select user roles.
 1. Click **Add user** to add the user to your organization. The user will receive
 an email with instructions on how to log in.
