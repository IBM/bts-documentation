<PageDescription>

**IBM Blockchain Transparent Supply™** APIs are available on Swagger. Each Swagger page provides a
link for users to authenticate to call the API endpoints.

</PageDescription>

## Authenticate to use the UI

To use the <a href={process.env.GATSBY_UI_URL + '/'}>IBM Blockchain Transparent Supply™ UI</a>, human users log in
with their IBMid that is registered with IBM Blockchain Transparent Supply™. An **Account Admin** for the organization
must register all user IBMids with IBM Blockchain Transparent Supply™ and assign the appropirate roles to individual
users.

## Authenticate to call APIs

The following links point to the IBM Blockchain Transparent Supply™ **production** zone APIs (to use the **sandbox**
zone APIs, navigate using the sandbox equivalents):

**IBM Blockchain Transparent Supply™ APIs:**

- <a href={process.env.GATSBY_UI_URL + '/ift/api/connector/swagger-ui.html'} target="_blank">Connector API</a> (upload data)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/converter/swagger-ui.html'} target="_blank">Converter API</a> (convert spreadsheets to upload data)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/outbound/api-docs/'} target="_blank">Trace API</a> (trace products)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/documents/api-docs/'} target="_blank">Documents API</a> (upload and share documents)

**Attention:** To authenticate to the IBM Blockchain Transparent Supply™ APIs as a human user, follow the
authorization steps on the respective Swagger page.
