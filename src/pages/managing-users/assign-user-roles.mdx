<PageDescription>

**IBM Blockchain Transparent Supply™** includes defined static roles that grant authorization to users, by registered <a target="_blank" href="https://www.ibm.com/account/us-en/signup/register.html">IBMid</a>,
to execute certain network tasks on behalf of their **organization**. Assigning roles enables an **Organization administrator** to easily control the level of access provided to each user in their organization.

</PageDescription>


## Prerequisites

When onboarding new users to IBM Blockchain Transparent Supply, an **Organization administrator** must assign one or more roles to each user. Before assigning roles, the following prerequisites
must be met:

* Your **organization** and your **Organization administrator** must be onboarded to IBM Blockchain Transparent Supply.
* Each user being onboarded must have a registered <a target="_blank" href="https://www.ibm.com/account/us-en/signup/register.html">IBMid</a>.

## Add or modify role assignments

An **Organization administrator** can add or modify user role assignments:

1. Log in to <a href={process.env.GATSBY_UI_URL + '/'} target="_blank">IBM Blockchain Transparent Supply™</a> as an **Organization administrator**.
1. Select the <a href={process.env.GATSBY_UI_URL + '/manageUsers'} target="_blank">**Users**</a> module.
1. Select the **Account users** tab.
1. To add roles for a new user, select the new user icon. Add the user and select the roles to assign.
1. To modify roles assigned to a user, select a user and click the edit icon. Select the roles to assign.

Note: Roles are not assigned to System IDs.
