<PageDescription>

**IBM Blockchain Transparent Supply™** provides APIs for registered and authenticated third-party applications to submit data requests.
For building applications, APIs provide simplicity of integration, a low barrier to entry, and rapid deployment using
industry standards. Only the applications that are authorized by the owner of the data can successfully extract that data.

</PageDescription>

## When to choose this method

Choose the **APIs** method of building applications when your top priority is speed and ease of deployment.

## About APIs

The **IBM Blockchain Transparent Supply™** APIs, available on Swagger, can be called by any authorized user to extract data related
to product trace, product insights and documents. The following **IBM Blockchain Transparent Supply™** APIs are provided to
support application development:

- <a href={process.env.GATSBY_UI_URL + '/ift/api/connector/swagger-ui.html'} target="_blank">Connector API</a> (upload data)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/converter/swagger-ui.html'} target="_blank">Converter API</a> (convert spreadsheets)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/outbound/api-docs/'} target="_blank">Trace API</a> (trace products)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/documents/api-docs/'} target="_blank">Documents API</a> (share documents)
- <a href={process.env.GATSBY_UI_URL + '/ift/api/visibility/api-docs/'} target="_blank">Insights API</a> (view metrics)

## Availability

**IBM Blockchain Transparent Supply™** APIs are currently available to all **IBM Blockchain Transparent Supply™** organizations.
