## Supported browsers

IBM Blockchain Transparent Supply™ currently supports the following web browsers:

* Google Chrome
* Mozilla Firefox
* Apple Safari
* Microsoft Edge

Use of any other web browser for IBM Blockchain Transparent Supply™ is
not guaranteed to provide complete functionality.
